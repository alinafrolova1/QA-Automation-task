For Windows and macOS:
1. Create a project folder and run the command in this directory: "npm install cypress --save-dev".
2. Run Cypress with a command: "npx cypress open".
3. Ensure that the cypress folder has been installed in the folder of your project.
4. Move loginData.json file to the cypress\fixture directory.
5. Move loginTest.spec.js file to the cypress\integration directory.
6. Move Pages folder to your project directory.
6. Navigate to the Cypress application and find the necessary test.
7. Run this test clicking on it.
